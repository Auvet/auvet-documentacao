# Backlog

## User Stories por Perfil de Usuário

### Administrador da clínica (ADM)

**ADM-01** – Como administrador, quero criar uma conta para minha clínica para que eu possa gerenciar pacientes, tutores e atendimentos.

**ADM-02** – Como administrador, quero cadastrar funcionários (veterinários/recepcionistas) para que eles possam usar o sistema com seus próprios logins.

**ADM-03** – Como administrador, quero listar funcionários cadastrados para visualizar e gerenciar a equipe da clínica.

**ADM-04** – Como administrador, quero editar dados de funcionários para manter as informações atualizadas.

**ADM-05** – Como administrador, quero excluir funcionários para remover registros desnecessários do sistema.

**ADM-06** – Como administrador, quero cadastrar tutores para que eles possam acessar os dados dos seus animais.

**ADM-07** – Como administrador, quero listar tutores cadastrados para visualizar e gerenciar os tutores da clínica.

**ADM-08** – Como administrador, quero editar dados de tutores para manter as informações atualizadas.

**ADM-09** – Como administrador, quero excluir tutores para remover registros desnecessários do sistema.

**ADM-10** – Como administrador, quero cadastrar animais (pacientes) para que eles passem a constar no sistema da clínica.

**ADM-11** – Como administrador, quero listar animais cadastrados para visualizar e gerenciar os pacientes da clínica.

**ADM-12** – Como administrador, quero editar dados de animais para manter as informações atualizadas.

**ADM-13** – Como administrador, quero excluir animais para remover registros desnecessários do sistema.

**ADM-14** – Como administrador, quero cadastrar consultas para registrar atendimentos realizados.

**ADM-15** – Como administrador, quero listar consultas cadastradas para visualizar e gerenciar os atendimentos da clínica.

**ADM-16** – Como administrador, quero editar consultas para atualizar informações de atendimentos.

**ADM-17** – Como administrador, quero alterar o status de consultas para acompanhar o andamento dos atendimentos.

**ADM-18** – Como administrador, quero excluir consultas para remover registros desnecessários do sistema.

**ADM-19** – Como administrador, quero cadastrar vacinas aplicadas para manter o histórico de vacinação dos animais.

**ADM-20** – Como administrador, quero listar vacinas cadastradas para visualizar e gerenciar o histórico de vacinação.

**ADM-21** – Como administrador, quero editar dados de vacinas para manter as informações atualizadas.

**ADM-22** – Como administrador, quero excluir vacinas para remover registros desnecessários do sistema.

**ADM-23** – Como administrador, quero visualizar métricas da clínica (número de animais, tutores, funcionários, consultas e vacinas) para ter uma visão geral do negócio.

**ADM-24** – Como administrador, quero configurar meu perfil pessoal para manter minhas informações atualizadas.

**ADM-25** – Como administrador, quero configurar os dados da clínica para manter as informações da clínica atualizadas.

### Funcionário (FUNC)

**FUNC-01** – Como funcionário da clínica, quero cadastrar um novo animal (paciente) para que ele passe a constar no sistema da clínica.

**FUNC-02** – Como funcionário da clínica, quero listar animais cadastrados para visualizar os pacientes da clínica.

**FUNC-03** – Como funcionário da clínica, quero cadastrar o tutor de um animal para que ele possa acessar os dados do seu pet.

**FUNC-04** – Como funcionário da clínica, quero listar tutores cadastrados para visualizar os tutores da clínica.

**FUNC-05** – Como funcionário, quero cadastrar consultas realizadas para um paciente para que o histórico médico fique sempre atualizado.

**FUNC-06** – Como funcionário, quero listar consultas cadastradas para visualizar os atendimentos da clínica.

**FUNC-07** – Como funcionário, quero cadastrar vacinas aplicadas para um paciente para que o histórico médico fique sempre atualizado.

**FUNC-08** – Como funcionário, quero listar vacinas cadastradas para visualizar o histórico de vacinação dos animais.

**FUNC-09** – Como funcionário, quero visualizar o dashboard com informações gerais da clínica para ter uma visão rápida do sistema.

**FUNC-10** – Como funcionário, quero configurar meu perfil pessoal para manter minhas informações atualizadas.

### Tutor (TUT)

**TUT-01** – Como tutor, quero acessar a aplicação com meu login para que possa acompanhar a saúde do meu animal.

**TUT-02** – Como tutor, quero visualizar o dashboard com informações gerais sobre meus animais.

**TUT-03** – Como tutor, quero visualizar os dados cadastrais dos meus animais para que eu tenha controle sobre as informações deles.

**TUT-04** – Como tutor, quero visualizar os agendamentos e consultas dos meus animais para que eu saiba quando levá-los à clínica.

**TUT-05** – Como tutor, quero visualizar o histórico de vacinas dos meus animais para que eu saiba quais já foram aplicadas e quais estão pendentes.

**TUT-06** – Como tutor, quero configurar meu perfil pessoal para manter minhas informações atualizadas.
